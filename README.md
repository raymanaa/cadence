# Cadence — Revenue call intelligence

Upload a sales call, get timestamped coaching. Sold to the rep, not the CRO.

Live: https://cadence.raymnz.com

## Stack
- Next 16 · static export · Cloudflare Workers + Static Assets
- Fraunces + Work Sans + IBM Plex Mono (next/font)
- framer-motion · lucide-react

## Portfolio context
Built by Rayen Manaa as part of an AI-SaaS portfolio. See `BRIEF.md` for the M0 design direction that was locked before scaffolding.

## Develop
```
pnpm install
pnpm dev
```

## Deploy

pnpm build
